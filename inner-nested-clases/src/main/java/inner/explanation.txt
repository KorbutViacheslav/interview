Внутрішній клас:
Ми можемо визначити клас у класі в Java, і ці класи називаються вкладеними класами.
Він використовується для логічної організації класів і досягнення інкапсуляції.
До зовнішніх членів класу (включаючи приватні) може отримати доступ внутрішній клас.

Існує 4 типи внутрішніх класів:
- Вкладений внутрішній клас
- Анонімні внутрішні класи
- Статичні вкладені класи
- Метод Локальні внутрішні класи.
(    -Nested Inner class
     -Anonymous inner classes
     -Static nested classes
     -Method Local inner classes)

 *Syntax:
class OuterClass {
    // Write the code
    class NestedClass {
        // Write the code
    }
}
