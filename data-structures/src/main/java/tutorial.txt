Що таке структура даних?
Структура даних — це сховище, яке використовується для зберігання та організації даних.
Це спосіб упорядкування даних на комп’ютері для ефективного доступу та оновлення.

Структура даних використовується не тільки для організації даних.
Він також використовується для обробки, отримання та зберігання даних.
Існують різні базові та розширені типи структур даних,
які використовуються майже в кожній програмі чи системі програмного забезпечення, які були розроблені.
Тому ми повинні добре знати структури даних.
